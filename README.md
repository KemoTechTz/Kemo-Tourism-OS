# Kemo Tours (Frontend-Only Showcase)

Kemo Tours is a premium tourism booking platform demo focused on Tanzania. It is a **frontend-only prototype** built with:
- HTML5
- Tailwind CSS (CDN)
- Vanilla JavaScript
- localStorage for persisted demo state

## Features
- Multi-page SPA-style experience (hash routing)
- Home, Packages, Package Details, Booking Flow, Customer Dashboard, Admin Dashboard
- 12 realistic Tanzania package records (mock data)
- Client-side search, filters, sort, wishlist, booking management
- Fake payment processing, booking reference generation, booking confirmation
- Currency toggle (USD/TZS), dark/light mode, toasts, mobile bottom nav, WhatsApp button

## Run
Open `index.html` directly in a browser.

## Deployment
Deploy as static files to Vercel/Netlify/GitHub Pages.
