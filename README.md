# Kemo Tourism OS (Frontend-Only Prototype)

Kemo Tourism OS is an enterprise-style, premium Tanzania tourism booking **operating system demo**.
It is intentionally **frontend-only** for presentations and static deployment.

## Stack
- HTML5
- Tailwind CSS (CDN)
- Vanilla JavaScript
- localStorage for persisted demo state

## Included Modules
- Overview
- Luxury Experiences
- Experience Details
- Booking Engine (5-step simulated flow)
- Guest Portal
- Operator Portal
- Admin Command Center

## Simulation Features
- Mock tourism inventory (12 premium experiences)
- Search, filters, sort, wishlist
- Simulated payment processing
- Fake booking reference generation (`KEMO-OS-#####`)
- localStorage persistence for bookings, status, payment status, and wishlist

## Run
Open `index.html` directly in a browser.

## Deploy
Deploy as static files on Vercel, Netlify, or GitHub Pages.
